# 包装设计指南V1.0 

### 前言

认识产品，从包装开始。

包装作为现代企业与消费者建立沟通的重要桥梁，在实现产品保护、方便储存与运输等功能的同时，还在极大程度上提升了产品的经济附加值。因此有一个良好的包装有助于提升商业、产品价值。


普通消费硬件产品，如何设计包装？一起探索吧。
